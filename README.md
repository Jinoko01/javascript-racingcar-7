# ✅ 체크포인트

- [ ] 자동차 이름을 쉼표를 구분자로 입력 후 cars 배열에 삽입
- [ ] 시도할 횟수 입력
- [ ] 0에서 9 사이의 무작위 값 추출
  - [ ] 4 이상일 경우 전진
  - [ ] 아니면 멈춤
- [ ] 매 횟수마다 이동 결과 출력
- [ ] 가장 많이 이동한 자동차를 winners 배열에 삽입
- [ ] 우승자를 쉼표를 구분자로 출력
- [ ] 사용자가 잘못된 값을 입력할 경우 "[ERROR]"로 시작하는 메시지와 함께 `Error`를 발생시킨 후 애플리케이션 종료
